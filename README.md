# Electronics Management System

🚀 A web-based Electronics Management System developed using ASP.NET to manage electronic products and user data.

## 🔥 Features
- User Registration and Login System
- Add New Products
- View Product List
- Manage Electronic Records (CRUD Operations)
- Simple and User-Friendly Interface

## 🛠️ Technologies Used
- ASP.NET (C#)
- HTML, CSS
- SQL Server

## 📂 Project Modules
- Authentication (Login & Registration)
- Product Management System
